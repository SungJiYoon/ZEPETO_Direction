# 3학년 2학기 [융합캡스톤디자인] Project
## 팀원 : 성지윤, 박다예, 이재우, 송태휘
## 본인 역할 : 제페토를 활용한 TypeScript 코드 작성(캐릭터 생성 및 이동), 길안내 기능

### 개발도구 : Unity
### 개발언어 : TypeScript, C#
### 플랫폼 : Mobile
### 프로그램 설명 : 제페토를 활용한 울산대학교 7호관 건물내 호실 길찾기 프로그램. 제페토 맵제작을 통해 울산대학교 7호관을 재현하고, 호실을 입력하면 화면상에 길을 안내하는 화살표가 뜬다. 화살표를 따라서 캐릭터를 이동시키며 실제로 내가 어떻게 호실을 찾아가면 되는지 알 수 있다. 7호관은 다른건물에 비해서 1층, 2층, 6층이 특이한 구조를 가지고 있기 때문에 호실을 안다고해도 처음 방문하는 사람은 미로처럼 느낄 수 있기에 실제로 어떻게 가면되는지 캐릭터 이동을 통하여 알 수 있다.
